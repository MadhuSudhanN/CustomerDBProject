# Customer Database Project
Creating a CRUD application for Customer with fields  
- Customer ID
- First Name
- Last Name
- Street
- Address
- State
- City
- Email
- Phone Number
 -  # API and FrameWork Needed :
- Database: Mysql
- Backend: JSp Servlet ,Spring, Hibernate
- Frontend : HTML, CSS, JS
